# Bibliothèque SFX — index (à lire à l'étape SFX)

> **Pour l'IA, à l'étape 6 (SFX) :** parcours cette liste, et pour **chaque moment qui mérite un son**
> (clic, apparition, transition, reveal, validation…), choisis le fichier dont la **description colle
> le mieux à ce qui se passe à l'écran**. Tu ne poses PAS tous les sons — tu choisis le bon, au bon
> endroit, avec parcimonie (mieux vaut 4-5 sons bien placés qu'un son sur chaque micro-élément).
> Règles de placement, volumes et recette de mix : `.claude/skills/motion-design/references/sfx-musique.md`.
> Tous ces fichiers sont **CC0** (domaine public), donc livrables/redistribuables.

## Clics

| Fichier             | À utiliser quand…                                                             |
| ------------------- | ----------------------------------------------------------------------------- |
| `click-mouse.mp3`   | un vrai clic de souris (curseur qui clique un bouton/lien à l'écran). Réaliste. |
| `click-double.mp3`  | un double-clic (ouvrir, lancer quelque chose).                                |
| `click-ui.mp3`      | un clic d'interface net et discret (case cochée, petit bouton, item de menu). |
| `select.mp3`        | on sélectionne / met en avant un élément (blip doux, un peu plus « produit »). |
| `tick.mp3`          | un tout petit tic sec (compteur, étape qui s'incrémente, coche rapide).       |
| `switch.mp3`        | un interrupteur / bascule on-off (activer une option, changer d'état).        |

## Feedback (positif / négatif)

| Fichier             | À utiliser quand…                                                             |
| ------------------- | ----------------------------------------------------------------------------- |
| `confirmation.mp3`  | une validation, un « c'est bon », un succès, une coche verte.                 |
| `error.mp3`         | une erreur, un « non », un mauvais choix, le truc à éviter (barre le mauvais). |

## Apparition d'un élément

| Fichier             | À utiliser quand…                                                             |
| ------------------- | ----------------------------------------------------------------------------- |
| `pop.mp3`           | un élément qui apparaît d'un coup (carte, bulle, valeur, icône). « Plop » net. |
| `pop-mouth.mp3`     | apparition plus douce / organique (variante de `pop`, moins synthétique).     |
| `drop.mp3`          | un élément qui « tombe » / se pose en place (petit impact court et léger).     |

## Transitions (whoosh)

| Fichier             | À utiliser quand…                                                             |
| ------------------- | ----------------------------------------------------------------------------- |
| `whoosh-fast.mp3`   | une transition RAPIDE, un swipe, un cut sec entre deux sections.              |
| `whoosh.mp3`        | une transition standard entre sections (whoosh moyen).                        |
| `whoosh-long.mp3`   | une transition plus ample / cinématique (mouvement de caméra, gros changement). |

## Montée de tension (riser)

| Fichier             | À utiliser quand…                                                             |
| ------------------- | ----------------------------------------------------------------------------- |
| `riser.mp3`         | une montée avant un reveal / une punchline. Le pic tombe à la fin du hook.    |
| `riser-dark.mp3`    | une montée plus grave / dramatique (sujet sérieux, suspense).                 |

## Impact / accent

| Fichier             | À utiliser quand…                                                             |
| ------------------- | ----------------------------------------------------------------------------- |
| `impact.mp3`        | un accent sec sur un mot fort ou un chiffre qui tombe.                        |
| `impact-boom.mp3`   | un gros impact grave / cinématique (arrivée d'un titre plein écran, choc).    |

## Notification / signal

| Fichier             | À utiliser quand…                                                             |
| ------------------- | ----------------------------------------------------------------------------- |
| `ding.mp3`          | une notification chaleureuse (cloche « ding-dong »), un rappel, un message.   |
| `notification.mp3`  | un petit signal cristallin (alerte discrète, notif d'appli).                  |

## Divers

| Fichier             | À utiliser quand…                                                             |
| ------------------- | ----------------------------------------------------------------------------- |
| `keyboard.mp3`      | un texte qui se tape (mot-clé du CTA, saisie d'une commande). Rogner à la durée. |
| `cha-ching.mp3`     | de l'argent (vente, gain, prix, « ça rapporte »). Tiroir-caisse.              |
