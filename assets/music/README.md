# assets/music — ta musique de fond

Ce dossier est **volontairement vide** : la musique n'est pas redistribuable (droits d'auteur),
on ne peut donc pas la livrer dans le repo. À toi de le remplir.

Dépose ici **ton MP3 de musique de fond** (le morceau que tu veux entendre sous tes Reels).

Déclare-le ensuite dans `/setup` (bloc E) pour que le système le connaisse. Volume par défaut :
**-26,5 dB** (assez présent pour porter le Reel, assez discret pour ne pas couvrir ta voix) —
tu peux l'ajuster dans `brand.config.json`.

La musique se pose à l'étape SFX + musique (étape 6), une fois le montage validé.
